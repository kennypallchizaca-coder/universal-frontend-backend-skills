# Assets

This directory stores static repository assets.

## Structure

- `readme/`: diagrams and visuals used by the public [README.md](../README.md)

## Conventions

- Prefer SVG for repository diagrams whenever possible.
- Keep filenames in `kebab-case`.
- Group assets by audience or surface instead of dropping files directly at the repository root.
- Treat these files as public-facing documentation artifacts.
